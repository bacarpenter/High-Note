**Ben Carpenters CS50 final project**

**The goal:** A web app built to allow poeple to shares notes of graditude and postitivity without fighting to be heard.

## 1. Main technologies:
 1. HTML/CSS
 2. Python (Flask)
 3.  SQL (SQLite3)
